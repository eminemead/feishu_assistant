# Example Skill

This is an example skill that demonstrates the Agent Skills Standard format.

## Purpose

This skill serves as a template for creating new skills. It shows:
- How to structure YAML frontmatter
- How to write clear instructions
- How to document examples and resources

## Instructions

When this skill is active, the AI should:

1. Follow the example pattern shown here
2. Adapt instructions to the specific task
3. Reference resources when available
4. Provide clear, actionable guidance

## Examples

**Example 1**: Basic usage
```
User: "Show me an example"
AI: [Uses this skill to demonstrate the pattern]
```

**Example 2**: With resources
```
User: "Use the template file"
```
